# O que ainda falta para SEO forte e indexação por IA

## 1. Falta conteúdo semântico forte para Google e IA

Hoje a homepage ainda está muito focada em:

- cards
- frases curtas
- marketing visual

Isso é bom para humanos.
Mas IA e Google precisam de:

- contexto semântico
- profundidade textual
- relações entre conceitos

### Hoje falta:

- textos institucionais mais ricos
- contexto de negócio
- explicações detalhadas
- páginas complementares

---

# Recomendação mais importante

## Criar páginas específicas

Hoje parece existir apenas uma landing.

Você precisa criar páginas SEO dedicadas:

| Página                       | Objetivo                |
| ---------------------------- | ----------------------- |
| /controle-de-ponto-digital   | Palavra-chave principal |
| /reconhecimento-facial-ponto | SEO específico          |
| /controle-de-jornada         | Intenção de busca       |
| /banco-de-horas-online       | Long tail               |
| /controle-de-ponto-mobile    | Mobile SEO              |

Isso é extremamente importante.

Hoje o Google provavelmente entende:

> Landing comercial

Mas ainda não entende:

> Autoridade em controle de ponto
