Política de Privacidade dos Aplicativos

Última atualização: 01 de julho de 2026

Introdução

A Auvix Tecnologia valoriza a privacidade dos usuários do aplicativo Auvix Ponto e está comprometida em proteger os dados pessoais tratados durante a utilização do aplicativo.

Esta Política de Privacidade explica quais informações são coletadas, como elas são utilizadas e quais são os direitos dos usuários.

1. Sobre o Aplicativo

O Auvix Ponto é um aplicativo destinado ao registro e gerenciamento de ponto eletrônico, permitindo a autenticação de colaboradores através de QR Code e demais mecanismos disponibilizados pela empresa responsável.

2. Dados Coletados

Durante a utilização do aplicativo poderão ser tratados os seguintes dados:

Dados de identificação

  • Nome
  • Identificador do usuário
  • Empresa vinculada
  • Dados necessários para autenticação

Essas informações são utilizadas exclusivamente para identificar o colaborador e permitir o acesso ao sistema.

Localização

Quando habilitado pela empresa responsável pelo colaborador, o aplicativo poderá coletar a localização do dispositivo no momento do registro do ponto.

A localização é utilizada exclusivamente para validação dos registros de ponto e não é utilizada para rastreamento contínuo.

Dados técnicos

Também poderão ser coletadas informações técnicas necessárias para o funcionamento do aplicativo, como:

  • Versão do aplicativo
  • Modelo do dispositivo
  • Sistema operacional
  • Identificadores técnicos necessários para segurança da aplicação

3. Permissões Utilizadas

Câmera

O aplicativo solicita acesso à câmera exclusivamente para leitura de QR Codes utilizados durante o processo de autenticação.

Nenhuma imagem ou vídeo é armazenado pelo aplicativo.

Localização

A localização pode ser solicitada para validar o local onde o registro de ponto foi realizado, quando essa funcionalidade estiver habilitada pela empresa responsável.

A localização não é utilizada para monitoramento contínuo do usuário.

4. Como Utilizamos os Dados

Os dados tratados são utilizados para:

  • Autenticar usuários
  • Validar registros de ponto
  • Registrar informações de presença
  • Aumentar a segurança da plataforma
  • Cumprir obrigações legais relacionadas ao controle de jornada

5. Compartilhamento de Informações

Os dados poderão ser compartilhados exclusivamente com:

  • A empresa responsável pela conta do colaborador
  • Provedores de infraestrutura utilizados para operação da plataforma
  • Autoridades competentes quando houver obrigação legal

A Auvix Tecnologia não vende informações pessoais dos usuários.

6. Armazenamento e Segurança

Os dados são armazenados utilizando medidas técnicas e organizacionais adequadas para proteger as informações contra acesso não autorizado, perda, alteração ou divulgação indevida.

7. Direitos do Usuário

O usuário poderá solicitar, quando aplicável:

  • Confirmação da existência de tratamento
  • Acesso aos dados pessoais
  • Correção de dados incompletos ou incorretos
  • Exclusão de dados quando permitido pela legislação
  • Informações sobre o tratamento realizado

As solicitações poderão ser encaminhadas pelos canais de contato informados nesta política.

8. Retenção dos Dados

Os dados serão mantidos pelo período necessário para cumprimento das finalidades descritas nesta política, bem como para atendimento de obrigações legais e regulatórias.

9. Crianças e Adolescentes

O aplicativo não é destinado a menores de 18 anos e não coleta intencionalmente dados pessoais de crianças.

10. Alterações nesta Política

Esta Política poderá ser atualizada periodicamente para refletir alterações legais, operacionais ou de funcionalidades do aplicativo.

A versão mais recente estará sempre disponível nesta página.
